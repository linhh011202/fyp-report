\subsection{Overview}
\label{sec:frontend_overview}

The Gateway Dashboard is a Vue.js 3 web application built with the Composition API and Vite, providing a modern management interface for the Speech Gateway API with real-time subscription management and authentication.

\subsubsection{Frontend Architecture}

Figure~\ref{fig:frontend_architecture} illustrates the layered architecture of the frontend application.

\begin{figure}[H]
\centering
\includegraphics[width=0.85\textwidth]{images/frontend_architecture.png}
\caption{Frontend Application Architecture}
\label{fig:frontend_architecture}
\end{figure}

The frontend follows a modern layered architecture with Vue 3 Composition API:
\begin{itemize}
    \item \textbf{Vue Router 4}: Client-side routing with navigation guards and beforeEach hooks
    \item \textbf{Views/Pages}: Page-level components using Composition API and composables
    \item \textbf{Composables}: Reusable composition functions (useAuth, useGlobalState, useSnackbar)
    \item \textbf{Components}: Reusable UI components like PlanCard with reactive properties
    \item \textbf{Services Layer}: API communication with fetch and Axios for authenticated requests
    \item \textbf{HTTP Interceptors}: Automatic token refresh and error handling middleware
    \item \textbf{Global State}: Reactive state management using Vue's provide/inject pattern
\end{itemize}
