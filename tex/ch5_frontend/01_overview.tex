\subsection{Overview}
\label{sec:frontend_overview}

The Gateway Dashboard is a Vue.js 2 web application that provides a management interface for the Speech Gateway API. This chapter covers the frontend architecture, component design, and integration with the backend API.

\subsection{Frontend Architecture}

Figure~\ref{fig:frontend_architecture} illustrates the layered architecture of the frontend application.

\begin{figure}[H]
\centering
\includegraphics[width=0.85\textwidth]{images/frontend_architecture.png}
\caption{Frontend Application Architecture}
\label{fig:frontend_architecture}
\end{figure}

The frontend follows a layered architecture:
\begin{itemize}
    \item \textbf{Vue Router}: Manages navigation and route guards.
    \item \textbf{Views/Pages}: Page-level components for each route.
    \item \textbf{Components}: Reusable UI components like PlanCard.
    \item \textbf{Services Layer}: API communication logic.
    \item \textbf{HTTP Client}: Axios with interceptors for token handling.
\end{itemize}
